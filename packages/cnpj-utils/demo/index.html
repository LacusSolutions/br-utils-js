CNPJ Utils

O seu utilitário para manipulação de CNPJ.

Biblioteca JavaScript completa para gerar, validar e formatar CNPJs com funcionalidades avançadas.

Instalação da biblioteca

  • NPM
  • Bun
  • PNPM
  • Yarn
npm install cnpj-utils check
bun add cnpj-utils check
pnpm add cnpj-utils check
yarn add cnpj-utils check
